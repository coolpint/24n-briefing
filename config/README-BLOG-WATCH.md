# 블로그 알림 운영 규칙

생성 스크립트: `scripts/blog_alert_watch.py`

## 고정 알림 형식
아래 4줄 템플릿만 허용한다.

1. `[메르의 블로그 업데이트]`
2. `제목: {게시글 제목}`
3. `요약: {핵심 요약 1문단}`
4. `링크: {원문 URL}`

형식이 하나라도 맞지 않으면 발송하지 않는다.

## 요약 품질 규칙
- RSS description 우선 사용, 없으면 본문 추출 사용.
- `출처/Unsplash/광고/구독/댓글/공유/관련기사` 등 노이즈 문구는 제거.
- 핵심 문장 1~2개로 요약(너무 짧은 문장 제외).
- 요약이 비정상이면 발송하지 않고 스킵한다.

## 재발 방지
- 템플릿 행 수(정확히 4줄) 검증을 통과한 경우만 전송.
- 포맷 불일치 시 상태를 갱신하지 않아 다음 실행에서 재시도 가능.
